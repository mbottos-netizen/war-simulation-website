# Operation Synthesis — Military Simulation Website

An interactive, multi-page research website created for the **"Future of Gaming" Interactive Website** assessment task (Industrial Technology, outcomes IND5-9 & IND5-10).

- **Area of impact:** Training & Simulation
- **Focus topic:** Military simulation (used by armed forces for training, mission rehearsal and war-gaming — *not* war video games)

## Pages
| File | Page |
|------|------|
| `index.html` | Homepage — introduces the topic & area of impact |
| `what-is.html` | What military simulation is & how it works (+ infographic) |
| `current-use.html` | How it's used today + embedded video |
| `impacts.html` | Positive, negative & social impacts |
| `global-impacts.html` | Cultural, environmental, local & global impacts |
| `future.html` | Emerging & future tech + conclusion/reflection |
| `references.html` | All sources acknowledged |

## Rubric features included
- Homepage + 5 content pages + references (exceeds "min. 4 additional pages")
- Interactive navigation menu (sticky, mobile-friendly toggle)
- Headings, subheadings, callouts and captions
- Written research + balanced analysis across all 10 research areas
- **Infographic** (LVC diagram + animated stat bars) on `what-is.html`
- **Embedded video** on `current-use.html`
- Interactive features: tabs, accordions, animated counters, scroll reveals, buttons & links
- Conclusion / final reflection on `future.html`
- Reference list on `references.html`
- Consistent tactical dark theme, readable fonts (Russo One + Chakra Petch)

## How to view / host
**Open locally:** double-click `index.html`, or run a local server:
```bash
cd war-simulations-website
python3 -m http.server 8000
# then open http://localhost:8000
```

**Get a shareable link (for submission):** push to GitHub and enable **GitHub Pages**
(Settings → Pages → deploy from branch), or drag the folder into Netlify Drop
(app.netlify.com/drop). Either gives you the "working website link" the task asks for.

## Before you submit
1. Replace the example reference links with the exact pages you used.
